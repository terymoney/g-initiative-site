# G Initiative Premium Landing Page

This is a rebuilt landing page starter for the G Initiative website.

## What is included

- Premium animated hero section
- 3-image cinematic background loop
- Slow luxury zoom-in / blur crossfade transition
- Real G Initiative logo treatment for dark and light surfaces
- Verified impact messaging based on the portfolio content
- Impact stats section
- About preview
- Focus/impact areas section
- Partner/investor CTA section
- Downloadable 2026 portfolio PDF in `public/assets/`

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in your terminal.

## Build test

```bash
npm run build
npm run preview
```
